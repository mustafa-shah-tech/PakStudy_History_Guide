## ✨ Features

| Feature | Description |
|---|---|
| 🔴 **Exam Alert Boxes** | Red-highlighted boxes flag every high-probability exam question |
| ✅ **Interactive Quizzes** | Checkpoint quizzes after each chapter with instant feedback |
| 📊 **Visual Learning** | Comparison grids, timelines, and formula / fast-facts blocks |
| 💾 **Auto-Save Progress** | Topic completion saved to `localStorage` — survives page refresh |
| 🌙 **Dark / Light Mode** | Toggle button + auto-detects your mobile system theme |
| 📱 **Mobile-First Design** | Hamburger menu, safe-area handling, tested from 320px → desktop |
| 🎨 **Pakistan Theme** | Crescent Green & Gold colour palette with glassmorphism UI |

## 📚 Content Covered

### Chapter 1 — The Awakening (1905–1906)
- Partition of Bengal (1905) & Its Annulment
- Simla Deputation (1906)
- Establishment of All India Muslim League

### Chapter 2 — Constitutional Reforms & Pacts (1909–1919)
- Minto-Morley Reforms (1909)
- Lucknow Pact (1916)
- Montagu-Chelmsford Reforms (1919)

### Chapter 3 — Path to Independence (1940)
- Pakistan (Lahore) Resolution (1940)

## 🚀 Getting Started

This is a **zero-dependency, single-file** web app. No build tools required.

```bash
# Clone the repo
git clone https://github.com/your-username/PakStudy_History_Guide.git

# Open in browser
open index.html
```

Or simply [view it live →](https://your-username.github.io/PakStudy_History_Guide/)

---

## 🌙 Theme Support

The guide **automatically matches your device/OS theme**:
- 🌑 **Dark mode** — deep forest greens & amber golds (default for dark OS)
- ☀️ **Light mode** — fresh sage greens & warm amber on cream (default for light OS)
- Toggle manually anytime using the **🌙 / ☀️ button** in the top bar (desktop) or the floating button (mobile)
- Your preference is remembered across sessions via `localStorage`
